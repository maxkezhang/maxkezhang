## 👋 About Me

Senior Software Development Engineer at Unity China, working on LLM and AI agent systems, CI/CD infrastructure, automated testing, observability, and developer tooling.

At Unity China, I have worked on infrastructure and quality engineering systems, including CI/CD platforms, automated testing tools, and workflow automation. More recently, my work has focused on LLM-based agents for software engineering workflows, task automation, and Unity-related agentic LLM fine-tuning with methods such as SFT and RL.

Previously, I worked on observability and AIOps research in the CodeWisdom group at [Fudan University's Software Engineering Lab](http://www.se.fudan.edu.cn/), with a focus on trace/log-based anomaly detection for microservice systems, graph neural networks, PU learning, cloud-native systems, Docker, and Kubernetes.

I am also an [Apache SkyWalking](https://skywalking.apache.org/) Committer, with contributions to the SkyWalking Python Agent, including profiling support, agent-side development, and distributed tracing/APM tooling.

My recent interests focus on AI agents for software engineering, especially agentic reasoning over APIs, tests, logs, traces, and developer workflows.
